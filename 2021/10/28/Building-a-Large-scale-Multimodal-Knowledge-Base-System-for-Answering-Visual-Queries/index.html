Building a Large-scale Multimodal Knowledge Base System for Answering Visual Queries

发表于2021-10-28|更新于2021-10-28
|阅读量:

论文：Building a Large-scale Multimodal Knowledge Base System for Answering Visual Queries

创新点：

  • 把MRF应用于知识库表示中，这样可以结合连续的图像特征变量和其它的离散变量。
  • 提出了一种声明式语言用于辅助检索
  • 采用了factor graph ，能够高效表示实体间的关系

摘要

  • 提出了一个知识库框架处理视觉查询任务，在遇到新的任务时不需要重新训练分类器
  • 把大规模MRF（马尔科夫条件随机场）用于KB表示，结合视觉、文本、其他结构性数据和它们之间的关系

Introduction

现有的基于NLP和视觉的查询任务都存在面对复杂问题查询时不能够识别的问题，一些文章提出将视觉识别任务放入一个异质性推理和推断更强的框架之中—————这样做的好处是在每次新的类型任务出现时不需要重新训练分类器

作者受NEIL模型和Markov Logic模型的启发，且侧重解决它们存在的问题，提出了自己的模型：

  • NEIL模型的缺点：在识别任务中测试场景受限，同时还缺乏一个配套的推理模型在不需要训练新的分类器的条件下扩展到更丰富的高级任务中

作者的解决方法：

  • Markov Logic模型的缺点：their testing scenario is limited by its small data size and the discrete representation

作者的解决方法：现有的基于文本的KB表示不能结合连续的视觉特征在一个概率框架中，这阻碍了我们表示更丰富的多模态信息。作者将MRF模型应用于KB表示中，这样就能在一个联合概率模型中混合离散的和连续的变量

效果：作者的KB有5亿个变量和四百万个参数，比Zhu等人的模型多出4个数量级同时训练时间还缩短了

Previous Work

joint Models in Vision:“In this work, we use an MRF representation for joint learning and inference of our data,casting MRF models into modern KB systems. In particular,we address the scalability challenge of large-scale MRF learning with our knowledge base construction system.”

Learning with Vision and Language:

  • Image and video captioning任务：为图像和视频生成一段小的文本描述
  • visual question answering任务：基于图像内容回答自然语言问题

captioning和问答任务都是在单一图像上进行，产生NLP输出。
我们的系统提供了一个单一的，连贯的框架，可以对一张或多张图像、文本元数据和其他形式的数据进行联合学习和推理

Knowledge Bases:一些简单的大规模KBs知识简单的缓存视觉内容和将这些内容通过超链接link到text上。他们缺乏一个推理框架来处理更多不同类型的视觉查询问题。

Learning the Large-scale KB System

Given our goal towards learning a real-world, large-scale MRF-based KB system, the biggest challenge we need to address here is efficient learning and inference.

knowledge base construction(KBC)的三个关键步骤：

  • 数据预处理

将原始数据预处理成结构化表示，尤其是关系数据库中的表格形式。每个数据库表格存储同一类实体

  • 生成因子图

  • 生成了一种声明式语言允许我们定义带有人工推理规则性质的factors

  • 同样的factor之间权值共享，权值共享方案在声明式语言中定义

  • 高效学习

我们为高效学习和推理构建了一个吉伯斯采样器能够处理多模态变量
